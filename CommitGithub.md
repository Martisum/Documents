# 如何把代码同步到GitHub???
> 顺便讲一下怎么在**vscode**里面预览markdown

> *其实直接按ctrl+k再按c就可以了*

git init

git add .

git commit -m 'test'

git remote add origin 你的远程库地址

git pull --rebase origin master

git push -u origin master

git status

上面的内容就是直接合并了……

你应该知道直接合并是什么意思
